# Colors

Four primitive color families. Each generates a full 11–13 step scale. ★ marks the primary production step for each family. All tokens reference these primitives — **never hardcode hex values in components**.

## Semantic Tokens

Always use semantic tokens in components — never raw primitive values. This allows brand updates without touching component code.

### Brand

| Token | Primitive | Hex | Usage |
|-------|-----------|-----|-------|
| `--color-brand-primary` | teal-500 | `#0B8FBF` | Primary CTAs, active states, hero, app icon, nav selected, in-progress status |
| `--color-brand-primary-dark` | teal-700 | `#084E63` | Hover state of primary, dark hero backgrounds |
| `--color-brand-primary-light` | teal-75 | `#DCF7FD` | Active card backgrounds, selected package tint |
| `--color-brand-accent` | gold-400 | `#DEB84D` | Star ratings, pricing, "Confirm & Pay" CTA, premium badges |
| `--color-brand-accent-dark` | gold-600 | `#A97D1E` | Gold text on light backgrounds |
| `--color-brand-heritage` | maroon-700 | `#6B2535` | National Day campaigns, Qatar-specific promotions, formal vendor docs only |

### Text

| Token | Primitive | Hex | Usage |
|-------|-----------|-----|-------|
| `--color-text-primary` | pearl-900 | `#1E2025` | All primary body text, headings |
| `--color-text-secondary` | pearl-600 | `#636A78` | Supporting text, descriptions, captions |
| `--color-text-tertiary` | pearl-400 | `#A8B0C0` | Placeholders, hints, timestamps, disabled labels |
| `--color-text-disabled` | pearl-300 | `#C8CDD8` | Disabled state text |

| `--color-text-link` | teal-500 | `#0B8FBF` | Link default |
| `--color-text-link-hover` | teal-700 | `#084E63` | Link hover |
